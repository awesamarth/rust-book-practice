# ows-touchid Demo

## Prerequisites
- Mac with Touch ID
- `ows-touchid` installed via `bun link`
- `bun` installed

---

## 1. Wallet Creation
```bash
ows-touchid wallet create --name "my-wallet"
```
Touch ID popup aayega. Authenticate karo — wallet create ho jaayega with addresses for 8 chains.

---

## 2. Wallet List
```bash
ows-touchid wallet list
```

---

## 3. Key Creation (for agent)
```bash
ows-touchid key create --name "my-agent" --wallet my-wallet
```
Touch ID popup aayega. Token save kar lo — sirf ek baar dikhega.

```bash
ows-touchid key list
```

---

## 4. Sign a Message (as owner, Touch ID)
```bash
ows-touchid sign message --wallet my-wallet --chain evm --message "hello world"
```

## 5. Sign a Message (as agent, no Touch ID)
```bash
OWS_AGENT_KEY="ows_key_..." ows-touchid sign message --wallet my-wallet --chain evm --message "hello world"
```

---

## 6. Send Transaction on Sepolia (as agent)

Make sure wallet has Sepolia ETH. Then:

```bash
OWS_AGENT_KEY="ows_key_..." bun run sign-and-send.ts
```

This will:
1. Build an unsigned tx (send 0.001 ETH to self)
2. Sign it via `ows-touchid` CLI using agent key (no Touch ID)
3. Broadcast to Sepolia

TX hash will be printed — verify on https://sepolia.etherscan.io

---

## 7. Wallet Deletion
```bash
ows-touchid wallet delete --wallet my-wallet
```
Touch ID popup aayega. Confirm karo — wallet delete ho jaayega.
